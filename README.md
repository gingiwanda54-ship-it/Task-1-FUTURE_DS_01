# 📊 Sales Data Analysis Dashboard

## 📌 Project Overview
This project focuses on analyzing business sales data to uncover insights that can drive better decision-making.

The goal was to go beyond charts and provide real business recommendations based on data.

---

## ❓ Business Questions
- Which products generate the most revenue?
- How do sales change over time?
- Which regions and categories are most profitable?
- Where should the business focus to grow faster?

---

## 🛠️ Tools Used
- Microsoft Excel → Data cleaning and preparation  
- Power BI → Dashboard creation and visualization  

---

## 📊 Dashboard Features
- KPI Cards (Revenue, Profit, Orders, Profit Margin)
- Sales trend over time (Line chart)
- Profit trend analysis
- Sales by category
- Profit by region
- Top-performing products table
- Interactive filters (Region, Category, Date)

---

## 📈 Key Insights
- Technology products generate the highest profit
- The West region is the most profitable
- Furniture category has low margins
- Sales show strong seasonal trends

---

## 🚀 Recommendations
- Focus on high-margin products (Technology)
- Reduce excessive discounting
- Improve performance in low-profit regions
- Leverage peak sales periods with marketing campaigns
